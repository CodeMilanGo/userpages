Synonyms - Python_ try-except translated to Golang_
(Go_ programming language).

Python_:

.. code-block:: python

  try:
      one()
      two()
      three()
  except:
      handler()

Go_:

.. code-block:: go

  func main() {
  	defer func() {
  		e := recover()
  		if e != nil {
  			handler()
  		}
  	}()
  	one()
  	two()
  	three()
  }

missing.
