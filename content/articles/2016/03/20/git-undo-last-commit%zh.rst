git復原最後一次的commit
#######################

:date: 2016-03-20T05:19+08:00
:tags: Linux, git
:category: Linux
:summary: git復原最後一次的commit
:adsu: yes

從 [1]_ 的回答：

.. code-block:: bash

  # 重設上次修改
  $ git reset HEAD~
  # 編輯並再次git add修改，完成後：
  $ git commit -c ORIG_HEAD

----
